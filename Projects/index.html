Structure

All HTML files contains 3 major components, the html, header, and body tag. The html is the overall part of the file which contains the head and the body of the file.

  • Header

    • This contains all condition, external files, and details that this html file should have that is not seen by the client

    Tag: head

  • Title

    • This is included in the header file which shows name the title of file on a window screen, or namely the title of the page

    Tag: title

  • Body

    • This is the second major component contained in the HTML file, this part contains all the information that is available to the clienet and is written with tags and text that is listed in section 2 and 3

    Tag: body
