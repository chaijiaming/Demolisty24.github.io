  • Heading Levels

    • There are various level of heading that you can use to make your page look better, there are 6 levels of heading, with level 1 being the largest and 6 is the smallest. The font size if changable through CSS

    1. This is a Main heading h1

      2. The tag for Main heading is h1

    2. This is a level 2 heading

      3. The tag for level 2 heading is h2

    3. This is a level 3 heading

      4. The tag for level 3 heading is h3

    4. This is a level 4 heading

      5. The tag for level 4 heading is h4

    5. This is a level 5 heading

      6. The tag for level 5 heading is h5

    6. This is a level 6 heading

      7. The tag for level 6 heading is h6

  • Paragraph

    • The tag for paragraph is p

    Example:

    This is a paragraph

  • Bold

    • The tag for bold is b

    Example:
    Bolded

  • Italic

    • The tag for italic is i

    Example:
    Italic

  • Notation

    • The tag for notation is sup

    Example:
    Notation E = mc2

  • Subscript

    • The tag for subscript is sub

    Example:
    Subscript V1

  • Line Breaks

    • The tag for line breaks is br /

    Example:
    Print
    line breaks

  • Horizontal Rules

    • The tag for horizontal is hr /

    Example:
    Print
    Horizontal Rules

  • Strong

    • The tag for strong is strong

    Example:
    Issac Newton co-created calculus and the Newton's law of physic.

  • Emphasis

    • The tag for emphasis is em

    Example:
    Johnson was the first to land on Mars.

  • Block Quote

    • The tag for block quote is blockquote

    Example:
    Goelz said more money needs to be spent to improve screening.

  • Quotation

    • The tag for quotation is q

    Example:
    Steven once said The world is flat.

  • Abbreviation and Acronym

    • The tag for abbreviation and acronym is abbr

    Example:
    Dr. Hannan is one of the best computer science professor at Pennsylvania State University.
    PSU is one the top engineering school in the United States.

  • Citation

    • The tag for citation is cite

    Example:
    Central Limit Theory explains how a sequence of indentical and independent random variables can tends to the standard deviation as the sequence grows to infinity.

  • Definition

    • The tag for definition is dfn

    Example:
    Mean is the average of the sum of all given numbers.

  • Address

    • The tag for address is Address

    Example:
    E600 Pollock Road, 3101
    State College, PA 16802
    szl5645@psu.edu

  • Insert and Delete

    • The tag for insert is del and the tag for delete is del

    Example:
    Studying for the test last minute was the best worst idea ever. (del before and ins after)
    Never Always study last minute. (ins before and del after)

  • No longer relevant but should not be deleted

    • The tag for not relevant but to be kept is s

    Example:
    Samsung Galaxy S7 Edge MSRP:
    Was $749
    Now only $450
