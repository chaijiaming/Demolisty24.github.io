Table

  • Basic Strucutre of a Table

    • To create a table, we would need to use the element table, which you should think of it as a the warpper or the overall part which contains the rows and data. Then inside the table element, you will need to use the tr element to create rows in the table and store data using the tag td between a tr tag. This allow you to create a table with rows of data

    Tag: table (first and wrap everything inside the table with the ending tag), tr (to create rows in the table), and td (to create data in the a row in the table).

    Example:
    An example of how a table code is written is:
    86 60 48
    52 48 70
    92 70 86

  • Table Heading

    • Instead of the the table that we saw above without any label or tag associating with the row and column, we can use something call table heading (th) to set a name for those columns and rows.

    Tag: th

    Example:
    An example of how a table with heading is written is:

    The scope in the code is used to identify whhether it is a heading for a column or for a row.
    Midterm 1 Midterm 2 Final
    CMPSC 465 86 60 48
    CMPEN 331 52 48 70
    ECON 102 92 70 86

  • Spanning Over a Rows or Column

    • For a particular data in the table, it is possible to have it span over a column or a row with the attribute colspan or rowspan

    • Row Spanning

      • You can span a data over to another row using the attribute rowspan followed by the number of row you want to span

      Attribute: rowspan

      Example:
      An example of using rowspan is:

      Task to do:
      9:00 10:00 11:00 12:00
      Saturday Sweeping Cooking Eating Laundry
      Sunday Grocery Cleaning

    • Column Spanning

      • You can span a data over another column using the attribute colspan followed by the number of column you want to span

      Attribute: colspan

      Example:
      An example of using colspan is:

      Task to do:
      9:00 10:00 11:00 12:00
      Saturday Sweeping Cooking Lunch
      Sunday Cleaning Watch TV

  • Long Table

    • In the case where we have a very long table, we can separate the table in the part of the header, body, and footer

    Tags: thead (for the header), tbody (for the body), tfoot (for the footer)

    An example of using these tags:

    Midterm 1 Midterm 2 Final
    CMPSC 465 86 60 48
    CMPEN 331 52 48 70
    ECON 102 92 70 86
    Average of Score 70 59 69
