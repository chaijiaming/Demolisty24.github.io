Links

Links are created through the a tag/elements, it must includes the attribute href to link to other address.

* An absolute URL is the full web adress which include the page doamain and follow path to the specific page.
* A relative URL is pages that is located within the same site or same folder as the file this specific page is located in, you do not need to specify the domain name but only the file name itself.

  • Link references to other sites

    • This is when absolute URL needed to be used, we might redirect the client to a site/page that is either not in the same page or file as the current page that the client is on. An example of this is that if you are on a yahoo news page and it redirect you to the news source page.

    A possible example of absolute URL is https://github.com/

    Example:
    Movie Reviews Site:
    • Empire
    • Metacritic
    • Rotten Tomatoes
    • Variety
    more specific paths
    • Guardian of the galaxy 2 review on rotten tomatoes
    • Cars 3 on imdb

  • Links to other pages on the same site/folder

    • This is when relative URL is needed to be used, we will redirect the user to another page on the same folder or the same site. There is no need to use the entire URL but only the modified path from the page file.

    Example:
    The table of content itself is a great example of the usage of relative URL
    * A possible path might be ../index.html or picture/subpage.html

  • Links to emails

    • The a element/tags not only allow linking to others sites but also the possibilty to be directed to an email address of another person through the addition attribute of mailto.

    Example:
    A possible look at how to write a mailto address is href="mailto:szl5645@psu.edu"
    Email me @ szl5645@psu.edu

  • Redirecting to links that opens in a new window

    • Normally with just the a tag/element, one would just be directly directed to that link on the same tab session, however, we can change that by setting the target attribute to _blank.

    Example:
    A possible look at how to open a link in a new windown is href="somelink" target="_blank"
    Github (This open github in a new tab/window)
