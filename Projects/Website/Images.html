Images

To add images to your page, you will need to use the img tag/element along with an URL to the image. This URL is usually a relative URL to a folder containing all of the images that you might use in the site

Note that an image tag is an empty tag, so this means that there is no ending tag and you need to include other attribute to the img tag to give more information to the img

Attributes

  • Source

    • Note that to first including an image on your site, not only do you need to include the img tag, but also the source link to the image. To link the source of the image to the page, in your img tag you will need to include the src attribute.

    Attribute: src

    An example of using the source attribute is img src="source path" /

  • Alternative

    • In the case where the picture is not available or is unable to connect to the source link, an alt attribute allows the user to see a text that describes that the picture was intended to be

    Attribute: alt

    An example of using the alt attribute is img src="source path" alt="some alternative name"

  • Title

    • The title attribute will simply provide additional information when hover over the picture

    Attribute: title

    An example of using the title attibute is img src="source path" title="some additional information" /

  • Height and Width

    • The image's height and width can be modified through the height and width tag

    Attribute: height (for height) and width (for width)

    An example of using the height and width atribute is img src="source path" height="some # in pixles" width="some # in pixles" /
    you can also just included only the height or the width by itself to modify that part without changing the other

Image File Types

There are various file type that is associated with each image.

  • JPEG (JPG): When a picture has alot of different colors in terms of either hue or saturation, it is recommended to have the picture's file type as JPEG.

  • GIF and PNG: When a picture has little colors or flat colors. (which is where there is an area with the same color all over), it is recommended to save the picture as a GIF or PNG. Note that subtle difference or shades of a particular color is not the same as that color itself)

  • SVG: This is a more relatively new image file type, it is made up of vectors, which means placing points on the grid and draw lines between those point and have color fill in the lines that have filled in. This file type would not be affected when changing its dimension size.

  • Animated GIF: This file type shows several frames of an image in sequence to show simple animations. Usually a loading screen logo is an animated gif, which is made up of several other individual gif and shown in sequence.

Transparency

We can also change how a picture look through the concept of transparency. There are 2 format that one can choose to create an inmage that is transparent, the transparent gif or the png.

  • It is recommended to save an image with straight edge and is 100% transparent as a gif with the transparency option selected

  • On the other hand, it is recommended to save an image with transparent part that has a diagonal or rounded edge or a semi-opaque or drop-shadow as a PNG
