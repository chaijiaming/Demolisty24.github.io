Lists

li is used after either an ordered list tae/element ol or an unordered list tag/element ul

  • Ordered List

    • The tag for ordered list is ol and we can create an entry with the usage of the tag list li

    Example:
    1. Step 1
    2. Step 2
    3. Step 3
    4. Step 4

  • Unordered List

    • The tag for unordered list is ul and we can create an entry with the usage of the tag list li

    Example:
    • Point 1
    • Point 2
    • Point 3
    • Point 4

  • Definition List

    • The tag for definition list is dl and we can create an entry with the usage of definition term dt and definition definition dd

    Example:
    Technology
    The practical application of knowledge especially in a particular area
    Odour
    Smell
    to perceive the odor or scent of through stimuli affecting the olfactory nerves

  • Nested List

    • The general idea is that you can have an ordered list or unordered list inside another list

    Example:

    • List 1

      • Point 1
      • Point 2
      • Point 3

    • List 2

      1. Step 1
      2. Step 2
      3. Step 3
